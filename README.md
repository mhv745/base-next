## Configuración

-   Actualizar variables de entorno

## Comandos

-   `yarn` - Instalar dependencias
-   `yarn dev` - Inicia el servidor de desarrollo
-   `yarn build` - Compila el proyecto
-   `yarn start` - Inicia el servidor de producción
